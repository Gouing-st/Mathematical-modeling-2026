2026年数学建模比赛的相关代码存储与共享

## 2026年APMCM亚太赛 C题 —— MATLAB 建模代码

| 文件 | 对应问题 | 内容 |
|---|---|---|
| `Q1_model.m` | Q1 选址 | 多准则选址：数据标准化（附录直取 / 三城调研内归一后融合）→ AHP 主观赋权（含 CR 一致性检验）→ 熵权客观赋权 → 50/50 组合赋权 → GRA 排序（主）+ TOPSIS 交叉验证 |
| `Q2_model.m` | Q2 容量配置与动态增长 | Logistic 增长 → 需求测算 → 预算约束下的一次性最优配置（网格搜索 λ*）→ 云端弹性补缺 → 利用率与生命周期成本，6张图 |
| `Q3_model.m` | Q3 空间功能分区 | 输入 Q1 选址 + Q2 容量/设备，输出五层楼面积分配、利用率验证，6张图 |
| `Q4_model.m` | Q4 十年投入产出与财务可行性 | 输入 Q1/Q2/Q3 结果，四流收入模型 + 成本模型 + 现金流/NPV/IRR/回收期 + 定价-补贴二维优化 + 折现率敏感性，6张图 |
| `Q5_model.m` | Q5 多维成效评估与分阶段运营方案 | 输入 Q1–Q4 全部结果，Leslie式梯队演化模型 + 熵权三合成指数 + 三阶段划分方案，6张图 |

各文件依赖链：Q1（选址）→ Q2（容量配置）→ Q3（空间分配）→ Q4（财务测算）→ Q5（综合评估），后一问的输入直接取自前一问的输出，详见各文件开头注释。
